# Citations & Bibliography

This page details the main references for Elastic Reasoning, Absolute Zero, and contextual classics in scalable reasoning, RLVR, and self-play. For in-text citations, see other documentation sections.

---

## Core Papers

- Xu, Y., Dong, H., Wang, L., Sahoo, D., Li, J., Xiong, C., “Scalable Chain of Thoughts via Elastic Reasoning”, arXiv:2505.05315v2, 21 May 2025.  
  [PDF](https://arxiv.org/abs/2505.05315)
- Zhao, A., Wu, Y., Yue, Y., Wu, T., Xu, Q., Lin, M., Wang, S., Wu, Q., Zheng, Z., Huang, G., “Absolute Zero: Reinforced Self-play Reasoning with Zero Data”, arXiv:2505.03335v2, 7 May 2025.  
  [PDF](https://arxiv.org/abs/2505.03335)

## Key Methods, Models, and Datasets

- Aggarwal, P., Welleck, S., “L1: Controlling how long a reasoning model thinks with reinforcement learning”, 2025.  
- Muennighoff, N., et al., “S1: Simple Test-time Scaling”, arXiv:2501.19393, 2025.
- DeepSeek-AI (Guo et al.), “Deepseek-r1: Incentivizing reasoning capability in llms via reinforcement learning”, arXiv:2501.12948, 2025.
- Gu, A., Rozière, B., et al., “CruxEval: A benchmark for code reasoning, understanding and execution”, ICML 2024.
- Jain, N., Han, K., et al., “LiveCodeBench: Holistic and contamination free evaluation of LLMs for code”, arXiv:2403.07974.
