## üß∞ Prerequisites

Ensure you have **Python 3.10+** installed. We recommend using **Python 3.12** for maximum compatibility.

Clone the repository:

```bash
git clone https://github.com/your-org/repo-name.git
cd repo-name
```

### Create and activate virtual environment _**macOS**_ & _Linux_
```
python -m venv .venv
source .venv/bin/activate 
````

### On Windows:
```
python -m venv .venv
source .venv\Scripts\activate
````

---

### 2. Installing Dependencies

You can install using **Poetry**, **uv**, **pip**, or from `setup.py`.

#### Option A: Using Poetry

```bash
# If poetry is not installed
pip install poetry

# Install dependencies
poetry install
```

> ‚úÖ **Note:** Poetry handles both dependencies and virtualenv management. If you use Poetry's built-in venv, skip manual `.venv` creation.

#### Option B: Using uv (fast pip-compatible installer)

```bash
pip install uv
uv pip install -r requirements.txt
```

#### Option C: Using pip (classic)

```bash
pip install -r requirements.txt
```

#### Option D: Using setup.py

```bash
pip install .
# or, for development mode
pip install -e .
```

---

### 3. Running the Project

Run the main entry point using one of the following:

#### With Poetry

```bash
poetry run python run.py
```

#### With uv (same as pip)

```bash
uv python run.py
```

#### Directly (if in .venv)

```bash
python run.py
```
